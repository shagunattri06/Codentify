# 🔍 Codentify – AI-Based Code Plagiarism Detection System

Codentify is a web-based platform developed as an AI/ML project to detect plagiarism between Java source code files.

Instead of comparing code line by line, Codentify analyzes the **Abstract Syntax Tree (AST)** of programs and uses a trained Machine Learning model to identify structural similarities, making plagiarism detection more accurate even if variable names or formatting are changed.

## Features

### Code Comparison
- Upload two Java source files
- Compare code using AST analysis
- Generate similarity score
- Predict whether the code is plagiarized
- Display confidence score for prediction

### AI/ML Engine
- AST-based feature extraction
- Random Forest Classifier
- Structural code similarity detection
- Fast prediction using a trained model
