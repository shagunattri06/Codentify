Technology

Understanding, not string-matching.

Codentify parses source code into an abstract syntax tree, then compares the resulting structure using graph-based similarity models. The result is a system that recognizes logic, not just characters.

  • Tokenization and AST parsing per language
  • Graph-based structural comparison
  • Confidence-scored similarity output

        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr.length - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }
