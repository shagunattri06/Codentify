About

Building Trust Through Intelligent Code Analysis

Codentify is an AI-powered code plagiarism detection platform developed as an academic machine learning project. It helps educators, institutions, and developers identify similarities in source code using structural analysis rather than simple text matching.

Our Mission

Promote academic integrity by providing accurate, fast, and explainable plagiarism detection.

How It Works

Users upload two source files. The backend preprocesses the code, extracts features, runs the trained AI/ML model, and returns similarity scores with visual reports.

Technology Stack

HTML, CSS, JavaScript, Python, Flask/FastAPI, Machine Learning, AST Parsing, Scikit-learn and Chart.js.

Key Features

  • Structural code comparison
  • AI-generated similarity score
  • Interactive charts
  • Responsive UI

Vision

To create a reliable AI-assisted plagiarism detection system for modern software education.

Academic Project

Developed by the Codentify team as a university AI/ML project showcasing intelligent software engineering.
