Privacy Policy

Last Updated: July 2026

Overview

Codentify is an AI-powered code plagiarism detection platform developed for educational and academic purposes. This Privacy Policy explains how uploaded source code and user information are handled while using our platform.

Information We Collect

  • Uploaded source code files
  • File names and programming language information
  • Basic browser and device information
  • Usage statistics for improving system performance

How Your Data Is Used

Uploaded files are processed solely to generate plagiarism detection results using our machine learning model. Files are not shared with third parties.

Machine Learning Processing

Uploaded code is analyzed using trained Artificial Intelligence and Machine Learning models to calculate similarity scores and detect potential plagiarism patterns.

Data Storage

By default, uploaded files are processed temporarily during analysis. Unless explicitly configured by an administrator, files are not permanently stored.

Security

Reasonable security measures are implemented to protect uploaded code and user information from unauthorized access.

Third-Party Services

Codentify does not sell or distribute uploaded code to third-party organizations.

Contact

For questions regarding this Privacy Policy, please contact the Codentify development team.
